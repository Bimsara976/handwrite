# COM763 Handwritten Digit Recognition

End-to-end multiclass ML pipeline and Streamlit demonstration for classifying 8×8 handwritten digit images.

## Local setup

```bash
python -m venv .venv
# Windows: .venv\Scripts\activate
# macOS/Linux: source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```

The repository includes the executed notebook, Streamlit app, exported pipeline, metadata, and evaluation artifacts.
